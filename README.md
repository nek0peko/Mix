## 关于主题

- Forked from: [wibus-wee/Mix-2.0.0](https://github.com/wibus-wee/Mix)

- 支持版本：Typecho v1.2.0-rc.2及以上

## 主题特色

短代码`基本全部`适配`handsome`主题

文章内页两种样式选择

多种样式配件选择

`不依赖`jQuery的Pjax，加载速度更快哦

全部使用*原生js*编写，不使用jQuery，优化加载速度

比较少见的主题布局风格

防剧透短代码 qwq
